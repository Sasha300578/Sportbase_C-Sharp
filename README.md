# Sport-base
Программа, выполняющая задачи хранения данных о спортсменах и тренерах, написанная на C#. 
Графический интерфейс основан на WindowsForms.

# Предпосылки к созданию приложения
ⅩⅩⅠ век становится временем информационных технологий, они 
приходят во все сферы нашей жизни, упрощая различные повседневные 
задачи. Однако в большинстве сфер деятельности человека всё еще 
информационные технологии не используются в полную меру. Во многих 
видах спорта работа с данными персонала и спортсменов происходит без 
использования технологий. Взаимодействие с документами и данными 
спортсменов - это неотъемлемая часть работы любой спортивной организации. 
Для всех спортивных мероприятий: соревнований, тренировок, спортивных 
сборов, необходима информация о спортсменах и их представителях. 
Необходимая информация о спортсмене может включать в себя: ФИО 
(фамилия, имя, отчество) человека, его возраст, паспортные данные, данные 
медицинского допуска, данные медицинской страховки от несчастного 
случая, ФИО тренера, спортивный разряд. А важная информация о тренерах 
может содержать: ФИО человека, его возраст, количество 
квалифицированных спортсменов, список спортсменов, занимающихся на 
данный момент, звание. Если не предоставить эти сведения вовремя, то можно 
оказаться без участия в важных мероприятиях спортивной подготовки, 
спортивного сезона.
Эта тема актуальна, так как, например, в таком спорте, как 
академическая гребля необходимо предоставлять все эти документы в 
бумажном виде для участия во всех спортивных мероприятиях, из-за этого 
возникают трудности совмещения работы с документами и тренировочного 
процесса. А также копии документов довольно часто теряются (как и 
оригиналы), из-за этого приходится тратить время на получение новых 
документов. Автоматизированные системы позволяют облегчить 
взаимодействие с данными людей. С их помощью можно хранить 
структурированную информацию, а также извлекать её удобным образом. Это 
позволяет сберечь время людей, их средства, силы, нервы, а главное время для 
получения нужной информации.

# Описание работы программного продукта
При запуске программы открывается главное меню, в котором 
пользователь выбирает как войти в программу, которое реализовано при 
помощи класса Form1.

<img width="203" alt="image" src="https://github.com/Sasha300578/Sportbase_C-Sharp/assets/113348429/26afbcb0-7953-40f1-a520-361cd94e3da4">

Если пользователь нажмёт кнопку «Гость», то будет открыто меню 
гостя. Если пользователь нажмёт кнопку «Администратор», то будет открыто 
окно авторизации администратора.

<img width="167" alt="image" src="https://github.com/Sasha300578/Sportbase_C-Sharp/assets/113348429/c1bbf8cb-083a-424e-a1fe-321a416a5aa3">

В меню гостя пользователь может нажать кнопку «Поиск спортсменов»,
тогда откроется окно поиска спортсменов. А также пользователь может
нажать кнопку «Поиск тренера», тогда будет открыто окно поиска тренеров.
Для того, чтобы вернуться на прошлое окно необходимо нажать на крестик в 
правом верхнем углу.

<img width="149" alt="image" src="https://github.com/Sasha300578/Sportbase_C-Sharp/assets/113348429/f494b37f-adeb-4781-8af7-21c552a513e4">

В окне поиска данных спортсмена пользователь может ввести фамилию, 
имя, отчество спортсмена, которого хочет найти, и нажать кнопку «Поиск».
Если данные такого спортсмена есть в базе данных, то будет открыто 
окно вывода данных спортсмена, будут выведены данные найденных 
спортсменов, иначе же появится надпись: «Спортсмены не найдены!». Если 
спортсменов найдено несколько, то появятся кнопки, при помощи которых 
можно листать спортсменов.

<img width="323" alt="image" src="https://github.com/Sasha300578/Sportbase_C-Sharp/assets/113348429/67741489-c74b-4697-a93c-9a4797cc7b81">

Поиск данных тренера работает аналогично поиску данных спортсмена.

<img width="142" alt="image" src="https://github.com/Sasha300578/Sportbase_C-Sharp/assets/113348429/eee10d6f-38fc-49cd-be2b-5a1915cc4a4c">

В окне авторизации администратора пользователь может ввести верные 
логин и пароль администратора и нажать кнопку «Войти», тогда откроется 
меню администратора. Если же данные окажутся неверны, то внизу окошка 
появится надпись, благодаря которой понятно, что введённые данные 
неверны.

<img width="152" alt="image" src="https://github.com/Sasha300578/Sportbase_C-Sharp/assets/113348429/fdf20fa9-4a32-443b-8121-43bdc8e8bdf1">

В меню администратора доступно 6 кнопок, каждая из которых при 
нажатии переходит на другую форму приложения. 
Кнопка «Добавить администратора» открывает окно для добавления 
нового администратора. 

<img width="155" alt="image" src="https://github.com/Sasha300578/Sportbase_C-Sharp/assets/113348429/f60deaa1-a533-4ba4-8d45-e7c145f331db">

В этом окне можно создать нового администратора. Необходимо дать 
согласие на сохранение данных.
Кнопка «Удалить администратора» открывает окно для удаления 
старого администратора.

<img width="143" alt="image" src="https://github.com/Sasha300578/Sportbase_C-Sharp/assets/113348429/c9e94f2c-586a-4934-a306-c3e5cd4d56e1">

В этом окне можно ввести данные администратора, которого нужно 
удалить, а потом нажав на кнопку, удалить. Необходимо дать согласие на 
удаление данных.
Кнопки «Поиск данных спортсмена» и «Поиск данных тренера» 
открывают окна поиска данных спортсмена и поиска данных тренера 
соответственно, аналогично тому, как это было для гостя, однако после 
нахождения данных для администратора доступна кнопка для перехода к 
редактированию данных в базе данных.

<img width="287" alt="image" src="https://github.com/Sasha300578/Sportbase_C-Sharp/assets/113348429/96208b39-3520-4db1-8fa1-4add3bf3b2ac">

При нажатии на кнопку «Редактировать данные тренера» будет открыто 
другое окно.

<img width="125" alt="image" src="https://github.com/Sasha300578/Sportbase_C-Sharp/assets/113348429/9839a78d-ef11-4297-9e11-4680a561608c">

В нём можно поменять некоторые сведения о тренере, который был 
открыт при поиске. Если данные не будут введены, но при этом нажать кнопку 
изменения данных, то справа от кнопки появится надпись: «Введите данные».
При нажатии кнопок будет вызвана одна из данных функций.
Изменения данных спортсмена происходят аналогично изменениям 
данных тренера.
Кнопки «Добавить нового спортсмена» и «Добавить нового тренера» 
открывают окна добавления данных нового спортсмена и добавления данных 
нового тренера соответственно. 

<img width="267" alt="image" src="https://github.com/Sasha300578/Sportbase_C-Sharp/assets/113348429/b7489a70-fcb4-4492-9719-f8b4ccbf4e5c">

В этих окнах можно создать нового спортсмена или нового тренера в 
базе данных. Для этого необходимо ввести все поля, а также дать согласие на 
сохранение данных, иначе кнопка «Создать» не сработает и появится
сообщение снизу под кнопкой, что необходимо сделать.
Добавление данных нового тренера или спортсмена реализовано 
аналогично добавлению нового администратора.
